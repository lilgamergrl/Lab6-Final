## Tutorial: Installing GitHub Copilot in VS Code

### Prerequisites
- Visual Studio Code installed on your computer
- A GitHub account
- GitHub Copilot subscription (free for students)

### Step 1: Get GitHub Copilot Student Access
1. Visit [GitHub Education](https://education.github.com/)
2. Click "Get benefits" or "Join GitHub Education"
3. Verify your student status with a school email or documentation
4. Once approved, GitHub Copilot will be included in your benefits

### Step 2: Install the GitHub Copilot Extension
1. Open Visual Studio Code
2. Click the Extensions icon in the left sidebar (or press `Ctrl+Shift+X` / `Cmd+Shift+X`)
3. Search for "GitHub Copilot"
4. Click "Install" on the official GitHub Copilot extension
5. Also install "GitHub Copilot Chat" for chat-based assistance

### Step 3: Sign In to GitHub
1. After installation, VS Code will prompt you to sign in
2. Click "Sign in to GitHub"
3. A browser window will open
4. Authorize VS Code to access your GitHub account
5. Return to VS Code—you should see a checkmark indicating successful setup

### Step 4: Verify Installation
1. Open a new `.java` file
2. Start typing a comment like `// function to calculate average`
3. Press Enter—Copilot should suggest code
4. Press `Tab` to accept suggestions
5. Open Copilot Chat with `Ctrl+Shift+I` / `Cmd+Shift+I` or click the chat icon

### Troubleshooting
- **No suggestions appearing?** Check the Copilot icon in the bottom-right status bar
- **"Copilot isn't active"?** Click the icon and select "Enable Globally"
- **Sign-in issues?** Try "Sign Out" then "Sign In" again from the Copilot menu
- **Student verification pending?** It may take 1-2 days for GitHub to verify student status
